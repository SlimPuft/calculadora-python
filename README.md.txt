# Calculadora en Python

Este proyecto es una calculadora simple desarrollada en Python que permite realizar operaciones matemáticas básicas desde la terminal.

## 🚀 Funcionalidades

* Suma
* Resta
* Multiplicación
* División
* Validación de división entre 0

## 🛠️ Tecnologías

* Python

## ▶️ Uso

Ejecutar:
python calculadora_simple.py

## 📌 Aprendizajes

* Uso de condicionales (`if`, `elif`, `else`)
* Entrada de datos con `input()`
* Manejo básico de errores
